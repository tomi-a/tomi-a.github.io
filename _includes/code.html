Game Making in the Unity Engine requires quite a bit of coding. This little practice project has 400 lines of C# code.

For example: here is the code for moving the player, checking if the player has movement, and activating the running animation at the appropriate time. 
        
myRigidbody.velocity = new Vector2(moveInput.x * runSpeed, myRigidbody.velocity.y);

isRunning = (Mathf.Abs(myRigidbody.velocity.x) > Mathf.Epsilon);

myAnimator.SetBool("isRunning", isRunning);
The PlayerScript can be seen in Github at: https://github.com/opafin
